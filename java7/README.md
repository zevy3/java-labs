# Лабораторная работа №7 - Многопоточность в Java

## Описание заданий

### Задание 1: Вычисление суммы элементов массива

**Вариант 1** (`Task1Variant1.java`):
- Создание двух потоков для вычисления суммы по половинкам массива
- Результаты складываются в главном потоке

**Вариант 2** (`Task1Variant2.java`):
- Использование `ExecutorService` и пула потоков
- Разделение массива на равные части
- Обработка каждой части отдельным потоком

### Задание 2: Поиск наибольшего элемента в матрице

**Вариант 1** (`Task2Variant1.java`):
- Создание отдельного потока для каждой строки матрицы
- Сравнение результатов в главном потоке

**Вариант 2** (`Task2Variant2.java`):
- Использование `ExecutorService` и пула потоков
- Разделение матрицы на равные части
- Поиск глобального максимума в главном потоке

### Задание 3: Склад с грузчиками (`Task3Warehouse.java`)

Симуляция работы склада с использованием многопоточности:
- 3 грузчика переносят товары между складами
- Максимальный вес за одну итерацию: 150 кг
- Использование `CyclicBarrier` для синхронизации грузчиков
- Использование `BlockingQueue` для хранения товаров

## Использованные концепции

- Многопоточность (Thread, Runnable)
- Пулы потоков (ExecutorService, Callable, Future)
- Синхронизация (CyclicBarrier, synchronized, wait/notify)
- Потокобезопасные коллекции (BlockingQueue)
